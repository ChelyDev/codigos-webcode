Váriaveis em JS: var, let. const. var: acessiveis dentro da função onde foram definidas. Se usada fora causa problemas de segurança e desempenho. var e let são iguais. let: let permite que ela tenha escopo de bloco, só é acessível dentro do bloco que ela é definida, mais recomendada para variáveis que precisam ser reatrinuídas, vomo contadores ou variáveis temporárias em loops. const: constante, não é variável e não muda. tipos primitivos: String, Number, Boolean, Undefined (variável sem valor definido), Null(variável null (nula)). variáveis estruturais (uma variável com diversas informações): Um array é uma estrutura de dados que armazena uma coleção ordenada de elementos. Pode conter diferentes tipos de dados, incluindo outros arrays. Podem ser acessados pelo índice.
